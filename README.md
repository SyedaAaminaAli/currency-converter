# 💱 Currency Converter

A modern Currency Converter web application developed during the People's Information Technology Programme (PITP) – MUET.

## Features

- 🌍 Real-time currency conversion
- 💱 Multiple currencies
- 🚩 Country flags
- 💲 Currency symbols
- 🔄 Swap currencies
- ⚡ Auto conversion
- ❌ Input validation
- 📱 Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
- ExchangeRate API
